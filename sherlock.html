       Sherlock: A Question Answering System                                                                                                                                                                    Back to Projects

 

 

Sherlock, named after IBM Watson's partner, is a question answering system developed by Tobin Yehle and me (Dasha Pruss) as the final project for Natural Language Processing. Sherlock answers questions about current event stories that were collected from the Canadian Broadcasting Corporation (CBC) web page for kids.
